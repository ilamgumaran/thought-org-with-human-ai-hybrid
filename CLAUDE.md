# Harmonized Intelligence Organization (HIO) Framework

## Project Purpose
This project contains the HIO Framework — a strategic framework for building organizations where human (organic) and AI (inorganic) intelligence work in harmony. It is designed for leaders setting up new organizations, particularly startup-style teams within larger enterprises.

## Key Concepts
- **Organic Intelligence**: Human cognition — creativity, judgment, emotional intelligence
- **Inorganic Intelligence**: AI/artificial cognition — analysis, pattern recognition, optimization
- **Harmonization**: Aligning both toward mutually fulfilling outcomes for all stakeholders
- **Cognitive Units**: Teams composed of humans + AI agents with defined roles

## Working with this project
- The framework is a living document — it should evolve with the user's thinking
- This is a strategic/philosophical project, not a codebase — treat it as structured thought
- When the user wants to develop specific sections deeper, create sub-documents in relevant directories
- Use deep thinking (ultrathink) for philosophical and strategic discussions
- The user values authentic win-win thinking, not zero-sum optimization
